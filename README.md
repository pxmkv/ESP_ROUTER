# ESP32-S3-ETH 单网口 DHCP 局域网示例（PlatformIO + Arduino）

这是给 **Waveshare ESP32-S3-ETH**（W5500 SPI 网口）准备的一个 **单网口有线局域网 DHCP server** 示例。

## 这个工程能做什么

- ESP32 自己固定地址：`192.168.50.1`
- 给接在交换机后的设备分配地址：`192.168.50.100 ~ 192.168.50.150`
- 提供一个简单状态页：`http://192.168.50.1/`
- 完全不使用 Wi‑Fi

## 这个工程不能做什么

- 不能直接当真正的双口路由器 / NAT 网关
- 不能只靠一个口同时做“上网 WAN + 下游 LAN 转发”
- 它更适合做一个 **单网口局域网控制节点 / DHCP 网关**

## 接线 / 使用方式

1. 烧录本工程。
2. 把 ESP32-S3-ETH 的 RJ45 口接到交换机。
3. 把电脑、工控机、开发板等多个设备接到同一个交换机。
4. 这些设备设成 DHCP 自动获取地址。
5. 打开串口监视器，波特率 `115200`。
6. 浏览器访问 `http://192.168.50.1/` 查看租约表。

## 板上 W5500 引脚

- CS: GPIO14
- RST: GPIO9
- INT: GPIO10
- MISO: GPIO12
- MOSI: GPIO11
- SCK: GPIO13

## 说明

这份代码是为了让你在 PlatformIO Arduino 环境下尽快跑通“交换机后面多个设备自动拿 IP”的需求。
如果你后面要升级成真正路由器，需要再加第二个网络接口（第二个 W5500 / USB 网卡 / Wi‑Fi 上行等），再做 NAT / 转发。
